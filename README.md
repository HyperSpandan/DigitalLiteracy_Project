## 📋 Module Summaries

### Module 1 – Digital Literacy Awareness (Task 1)
Created a one-page visual infographic covering five areas: what digital literacy is, safe internet practices, useful digital tools for students, email etiquette, and professional online presence. Tool used: HTML/CSS (browser-based design).

### Module 2 – Professional Digital Portfolio (Task 2)
Set up accounts on GitHub, LinkedIn, and Kaggle. Created a GitHub Profile README with name, branch, and learning goals. Filled in LinkedIn Education section with current degree details.

### Module 3 – Coding & Collaboration Platforms (Task 3)
Completed a beginner challenge on HackerRank (Python Introduction). Created a 5-question Digital Literacy Awareness Quiz using Google Forms, linked to a Google Sheets response tracker.

### Module 4 – Email Etiquette (Task 4)
Drafted two professional emails: one requesting an assignment extension and one expressing interest in a summer internship. Created a Social Media Do's and Don'ts checklist for college students.

### Module 5 – Cybercrime Awareness (Task 5)
Wrote a 240-word case study on UPI Payment Fraud detailing how it occurs step-by-step. Created a 10-point "Stay Safe Online" checklist including 2 UPI-specific tips and official reporting channels (cybercrime.gov.in, helpline 1930).
